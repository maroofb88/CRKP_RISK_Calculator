# CRKP Risk Calculator

Web-based calculator for predicting Carbapenem-Resistant *Klebsiella pneumoniae* (CRKP) infection risk using machine learning.

## Features
- Risk prediction based on demographic and laboratory data
- Clinical decision support
- Research use only

## Deployment
Deployed on Streamlit Cloud: [https://crkp-risk-calculator.streamlit.app](https://crkp-risk-calculator.streamlit.app)

## Model Details
- Algorithm: XGBoost
- Training data: 5,780 patients
- Test data: 1,445 patients
- AUROC: 0.613
- AUPRC: 0.263

## Important
⚠️ **For research use only.** Not for clinical decision-making without professional supervision.
